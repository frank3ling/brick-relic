# Contributing to BrickRelic

Thank you for your interest in BrickRelic! Please note that this repository is now **closed to external contributions**.

---

## 🚫 No External Pull Requests / Merge Requests

We do **not** accept external pull requests, merge requests, code contributions, or catalog submissions at this time. Any opened pull requests will be closed without merge.

If you have:
* **Bug Reports**: Please open a GitHub Issue describing the bug, including steps to reproduce, device configuration, and logs (if available).
* **Feature Ideas**: You are welcome to open a GitHub Issue with your ideas or suggestions.
* **Personal Customizations**: Since BrickRelic is licensed under the **GNU Affero General Public License (AGPL-3.0)**, you are free to fork the repository, modify the code, and run your own version, provided that you comply with the AGPL-3.0 copyleft requirements (including releasing the source code of your modifications if you host it over a network).

---

## 📋 Quality Checklist (For Internal Maintenance)

For the maintainer(s) of BrickRelic, all updates must satisfy:
- [ ] Code compiles successfully (`npm run build`).
- [ ] Linter reports no warnings or errors (`npm run lint`).
- [ ] All unit tests pass successfully (`npm run test`).
